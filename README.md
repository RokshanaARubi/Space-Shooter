# 🚀 Space Shooter Game (OpenGL)

A 2D Space Shooter game developed as a university group project using **C++** and **OpenGL**. The game features player movement, enemy spaceships, shooting mechanics, collision detection, scoring, and multiple gameplay elements.

> **Note:** This was a collaborative academic project completed by a team of five members. This repository showcases the final project and my contribution to the development.

---

## Project Overview

The objective of this project was to design and implement a simple arcade-style space shooter game while learning computer graphics concepts using OpenGL.

Players control a spaceship, avoid enemy attacks, destroy enemy ships, and try to achieve the highest possible score.

---

## Features

- Player spaceship movement
- Shooting mechanics
- Enemy spaceship generation
- Collision detection
- Score system
- Game Over screen
- Keyboard controls
- 2D graphics using OpenGL

---

## Technologies Used

- C++
- OpenGL
- GLUT / FreeGLUT

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/RokshanaARubi/Space-Shooter.git
```

2. Open the project in your preferred C++ IDE.

3. Make sure OpenGL and GLUT are installed.

4. Build and run the project.
